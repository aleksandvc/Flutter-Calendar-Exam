# flutter_calendar_exam

The final exam of Sirma's Flutter course.
!!! IMPORTANT: Should be tested only on android devices or emulators(API 33 and above)
Steps to test:
1. Clone the repo
2. Open VS Code
3. In terminal:
    1. go to the project root folder
    2. flutter clean
    3. flutter pub get
    4. cd ios
    5. pod install
    6. cd ..
    7. flutter run
4. Test
